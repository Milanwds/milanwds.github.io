Note concernant les préfectures

Si le préfecture doit passer entièrement dans le cacdre : si ce n'est pas le cas, n'hésitez pas à utilisez des initiales ou des mots tronqués (exemple : "SOUS-PREF DE SAINT-JULIEN", etc.)

Pour les grandes villes avec arrondissements, utiliser ce format : "Préfecture de police XX", XX correspondant à l'arondissement.
